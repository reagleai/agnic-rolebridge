## 2024-05-23 - [Setup & Profile Form Length Validation]
**Vulnerability:** SetupPage and ProfilePage allowed submitting textareas (resume text, job description) with virtually unlimited input length.
**Learning:** Unrestricted input fields on the client-side can allow arbitrarily large payloads to be sent to the backend, causing processing delays, memory bloat, or outright DoS if the payload is passed along to third-party services or LLMs without internal constraints.
**Prevention:** Apply defensive maxLength attributes in HTML textareas and enforce programmatic boundaries in submission handlers before firing backend requests. Also remember to ensure backend defense in depth validation.
